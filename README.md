# 🛡️ Cybersec Toolkit

A collection of advanced cybersecurity tools built in Python.
For educational and authorized use only.

---

## 🔧 Tools

### 🔍 NetScout — Network Scanner
- Port scanning (multi-threaded)
- OS fingerprinting via TTL
- Banner grabbing
- CVE vulnerability mapping (30+ known CVEs)
- Export to JSON / CSV
- Subnet scanning (CIDR)

### 🔐 Security Toolkit
- **Log Analyzer** — Apache / Nginx / SSH brute-force detection
- **Password Analyzer** — entropy, crack time, weakness detection
- **Password Generator** — random, passphrase, PIN, hex key

---

## 🚀 Installation
```bash
git clone https://github.com/YOUR_USERNAME/cybersec-toolkit.git
cd cybersec-toolkit
python3 netscout/netscout.py
python3 security-toolkit/security_toolkit.py
```

No pip installs needed — pure Python stdlib.
